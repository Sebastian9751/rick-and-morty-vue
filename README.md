# Rick And Morty WEB

#### This Vue app has been developed using TypeScript and The Rick and Morty API


### Dependencies
```bash
# To install dependencies, run this command in the root:
$ npm install
```

## Running the Server
```bash
# For development:
$ npm run dev
```
